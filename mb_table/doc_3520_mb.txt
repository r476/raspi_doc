Timer Type Meaning

Common part (in all modes):
  Bits  0- 6: Year (offset from 2000 => 2000-2127)
  Bits  7-10: Month (1-12)
  Bits 11-15: Day (1-31)
  Bits 16-31: Time (number of minutes from midnight => 0-1439)
    Year,Month,Day,Time = first start of timer (N/A in OFF mode)
  Bits 32-44: Duration (in minutes => 1-1439) (N/A in OFF mode)
  Bits 45-47: Mode
    0 = OFF
    1 = Once
    2 = Daily
    3 = Weekly
    4 = Monthly
    5 = Monthly (in day of week)
    6 = Period
    7 = N/A (OFF)

In OFF, Once, N/A mode:
  Bits 48-63: N/A

In Daily Mode:
  Bits 48-57: Repeat period (in days => 1-1023)
  Bits 58-59: Weekend mode:
    0 = including weekend days
    1 = except weekend days
    2 = postpone weekend days (to Monday)
    3 = N/A
  Bits 60-63: N/A

In Weekly Mode:
  Bits 48-53: Repeat period (in weeks => 1-63)
  Bit     54: on Monday
  Bit     55: on Tuesday
  Bit     56: on Wednesday
  Bit     57: on Thursday
  Bit     58: on Friday
  Bit     59: on Saturday
  Bit     60: on Sunday
  Bits 61-63: N/A

In Monthly Mode:
  Bits 48-51: Repeat period (in months => 1-15)
  Bits 52-56: Number of day in month (1-31)
  Bits 57-63: N/A

In Monthly (in day of week) Mode:
  Bits 48-51: Repeat period (in months => 1-15)
  Bits 52-54: Number of week in month (1-5)
  Bits 55-56: N/A
  Bit     57: on Monday
  Bit     58: on Tuesday
  Bit     59: on Wednesday
  Bit     60: on Thursday
  Bit     61: on Friday
  Bit     62: on Saturday
  Bit     63: on Sunday
    Note: currently only one day of week can be selected.
    Please refer to manual.

In Period Mode:
  Bits 48-63: Repeat period (in minutes => 1-65535)
